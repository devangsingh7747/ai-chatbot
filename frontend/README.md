# AI Chatbot 🚀

A Full Stack AI Chatbot application built using Java, Spring Boot, React, and MySQL.

> ⚠️ Project Status: Under Active Development

This project is currently being developed and improved continuously. New features, UI enhancements, backend improvements, and AI capabilities will be added over time.

---

## 📌 About The Project

The goal of this project is to build a modern AI-powered chatbot platform that supports persistent conversations, chat history, intelligent responses, and a clean user experience.

The application uses:

* Java
* Spring Boot
* MySQL
* React
* Vite
* REST APIs

The project follows a full-stack architecture with separate frontend and backend modules.

---

## ✅ Features Implemented

### Backend

* Spring Boot REST API
* Chat message processing
* MySQL database integration
* Conversation-based chat storage
* Chat history retrieval
* Persistent conversation IDs
* New chat session support

### Frontend

* React-based UI
* Modern chatbot interface
* Real-time messaging
* Conversation persistence after refresh
* Automatic chat history loading
* Responsive chat layout
* New Chat functionality

## 🚀 Running The Project

### Backend

```bash
cd backend
mvn spring-boot:run
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
